# **App Name**: Twin Response

## Core Features:

- AI Twin Messaging: Leverage Gemini 1.5 to provide chat responses based on portfolio data. LLM as tool
- Profile display: Display key profile details with links to Github, LinkedIn, and email.
- Project Showcase: Showcase projects with key metrics and associated tech stack.
- Career Timeline: Display the career history
- Resume link: Add an immediate link to resume, prominently displayed

## Style Guidelines:

- Primary color: Vibrant cyan (#06B6D4) to represent energy and technology.
- Background color: Dark, desaturated blue-gray (#0A0A0A) for a modern and professional feel.
- Accent color: A shade of purple (#A855F7) to add a layer of sophistication.
- Body and headline font: 'Inter', a grotesque sans-serif, for a machined, modern look.
- Use minimalistic icons for categories such as projects and career history.
- Implement a bento grid layout to display information in a structured and visually appealing manner.
- Subtle animations on hover to provide engagement. Gentle transitions for modal openings and closings.
